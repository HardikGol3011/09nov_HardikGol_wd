(9) How can the style/class of an element be changed?

Ans: we can use style.property for change in style

Ex : document.getElementById("id").style.property = new_style

We can also use classList property with add(), remove() and toggle() method.

Ex : document.getElementById("id").classList.add("classname")

Ex : document.getElementById("id").classList.remove("classname")

Ex : document.getElementById("id").classList.toggle("classname")
