Ans: ViewState : View state can only be visible from a single page and not multiple pages.

Information is stored on the clientâ€™s end only.

(8) What is === operator?

Ans: === is called Comparison Operator.

=== means equal value and equal data type.

Ex : 1

a = 50;

b = 50;

console.log(a===b);

then result is true.

Ex : 2

a = 50;

b = "50";

console.log(a===b);

then result is false.
