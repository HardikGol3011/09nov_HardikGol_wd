(2) What is a Bootstrap Container, and how does it work?

ANS : Container are basic layout in bootstrap and grid system.

there are 2 types of container in bootstrap (1) .container (2) .container-fluid.

.container have some margin while .container-fluid have 100% width.

container are most used to make layout in bootstrap

in container we have 2 classes to make layout (1) .row (2) .col

for make layout with bootstrap we can compalsory put .row class under container and compalsory put .col class under row.

Ex :

            <div class="container">
                <div class="row">
                    <div class="col"></div>
                    <div class="col"></div>
                    <div class="col"></div>
                </div>
            </div>
with row class we can get horizantal portion and with col class we can get vertical portion.

in bootstrap grid system maximum 12 column we can set.
