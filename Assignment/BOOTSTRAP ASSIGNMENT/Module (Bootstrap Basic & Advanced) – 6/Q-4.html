(4) What do you know about the Bootstrap Grid System?

ANS : The Bootstrap Grid System is used for layout, specifically Responsive Layouts.

in bootstrap grid system we can set maximum 12 column.

in bootstrap grid system we have 2 classes to make layout (1) .row (2) .col

for make layout with bootstrap grid system we can compalsory put .row class under container and compalsory put .col class under row.

with row class we can get horizantal portion and with col class we can get vertical portion.

in bootstrap grid system content should be placed in column.

Ex :

            <div class="container">
                <div class="row">
                    <div class="col">content</div>
                    <div class="col">content</div>
                    <div class="col">content</div>
                </div>
            </div>
if we want 3 eqaul column in row then we should use .col-4.

Ex : 
            <div class="container">
                <div class="row">
                    <div class="col-4"></div>
                    <div class="col-4"></div>
                    <div class="col-4"></div>
                </div>
            </div>
