(1) What is the difference between Bootstrap 4 and Bootstrap 5

ANS : in Bootstrap 4 Container Size is 1140px while in Bootstrap 5 Container size is 1320px.

in Bootstrap 4 there is Five Class prefixes, i.e col-xl, col-lg- , col-md- , col-sm-, col- .

in Bootstrap 5 there is Six Class prefixes, i.e col-xxl, col-xl, col-lg-, col-md- , col-sm-, col- .

In bootstrap 5 some classes remove which is available in the bootstrap 4 versions.

The form controls in Bootstrap 5 will have a significantly more consistent appearance and feel across all browsers due to their custom design.

Bootstrap 4 compatible with jumbotron while Bootstrap 5 not compatible with jumbotron.

in Bootstrap 4 There is no off-canvas element while in Bootstrap 5 The offcanvas component is now available.

Bootstrap 4 have fewer CSS properties while Bootstrap 5 have more CSS properties.

in Bootstrap 4 Columns can be positioned relative to one another while in Bootstrap 5 Columns cannot be positioned relative to one another.

Bootstrap 5 has its own set of SVG icons while in Bootstrap 4 we can get SVG icons from font-awesome.
