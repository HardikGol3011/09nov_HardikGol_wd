(2) How to Apply CSS Using JQuery, How to Add Class and Remove Class in Jquery , JQuery Animation?

Ans: Apply CSS Using jQuery :
    
$(document).ready(function(){
  $("button").click(function(){
    $("p").css({"background-color": "orange", "font-size": "20px"});
  });
});    

Add Class and Remove Class :

Syntax:

Adding a class:

$('selector').addClass(class_name);

Removing a class:

$('selector').removeClass(class_name);

JQuery Animation :

for jQuery Animation we can use animate() method.

Ex :

$("button").click(function(){
    $("div").animate({
      left: '150px',
      opacity: '0.3',
      color: 'blueviolet',
      background-color: 'purple',
      height: '180px',
      width: '250px'
    });
  });
