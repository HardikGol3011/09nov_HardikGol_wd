(3) Semantic element in HTML5?

 Ans :  an element that clearly describes its proper meaning 
        to both the browser and the developer is called 
        Semantic element.

the following elements are semantic elements :

<header> <footer> <nav> <article>
<aside> <form> <table> <section>
<figure> <figcaption> <mark> <details>
